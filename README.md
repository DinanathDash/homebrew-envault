# Homebrew Envault

Custom homebrew tap for the [Envault CLI](https://github.com/DinanathDash/Envault).

## Installation

```bash
brew tap DinanathDash/envault
brew install envault
```

## Maintenance

This repository is automatically updated by the primary [Envault repository](https://github.com/DinanathDash/Envault) via GitHub Actions and GoReleaser.

The `Formula/envault.rb` file is generated and updated automatically whenever a new version is released.
